# .dotfiles

1. Clone this repository
2. Run `install.sh`
3. Open up new window to initiate `zsh` shell

## Update Neovim

- :PlugInstall :PlugClean :UpdateRemotePlugins

## CocInstall

cd ~/.config/nvim/plugged/coc.nvim
yarn install
